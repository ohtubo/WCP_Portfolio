# TRPGにはまろう!

## サイト概要
TRPGシナリオ専用投稿サイト、シナリオ検索サイト

## サイトテーマ
TRPGのシナリオ投稿サイト

## テーマを選んだ理由
TRPGのサイトで個人でシナリオを投稿している人や紹介ブログ
等はあるが、本格的なシナリオ投稿検索サイトは見当たらない
と感じたため作成してみたいと考えた。

## ターゲットユーザ
作ったTRPGシナリオを公開したい人。
TRPGシナリオを探している人。

## 主な利用シーン
シナリオを投稿する。シナリオを探す。
